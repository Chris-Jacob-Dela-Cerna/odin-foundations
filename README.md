# html-foundations

A personal learning repository built alongside The Odin Project — Foundations. Holds practice, exercises, and assignments from HTML, CSS, and JavaScript as I work through the course.

---

## Structure

```
html-foundations/
├── html-basics/
├── css-basics/
├── javascript-basics/
└── README.md
```

Each folder holds whatever I build while learning that topic — TOP-required exercises and assignments, plus things I made up on my own to test a concept. No further nesting, just the work.

---

## On the practice projects

Some files here are assignments TOP explicitly asks for. Others are self-directed — built the moment a new concept clicked and I wanted to try it on something real.

---

## What this repo is not

Not a portfolio. Not polished. This is where I learn the concept before I build the thing that's supposed to look good. Projects that outgrow "practice" — like Recipes — get their own repo.
